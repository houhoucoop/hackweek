# React in Vue: Bring Backstage into Rancher

## Overview

Integrating React with Vue to create a Backstage extension

## Project Setup

1. set up Rancher UI extension (Vue@3)
```
yarn install
```
```
API=<Rancher Backend URL> yarn dev
```

2. set up Backstage application
```
cd pkg/backstage
```
```
yarn install
```
```
yarn dev
```
